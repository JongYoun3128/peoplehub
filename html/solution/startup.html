FAQ

  • 유연근무제를 도입하고 싶습니다. 도움을 주실 수 있나요?

    물론입니다. 이미 30만개가 넘는 기업과 사업장들이 피플허브와 함께 더 나은 근무환경을 만들었습니다. 피플허브는 스타트업에 알맞은 유연근무제 도입에 최적화된 솔루션입니다. 인력관리 트레이닝부터 유연근무제 도입까지 모두 도와드리겠습니다.

  • 연차 관리를 자동으로 할 수 있을까요?

    네. 피플허브는 연차 관리를 웹은 물론 모바일 앱에서도 제공합니다. 입사일 혹은 회계연도 기준으로 맞게 자동으로 연차를 발생시키고, 잔여 연차를 실시간으로 관리할 수 있습니다 (연차 촉진)

  • 출퇴근을 기록하지 않는 기업도 피플허브를 활용할 수 있나요?

    네. 피플허브를 사용하는 많은 기업들이 출퇴근을 기록하지 않고, 근무일정과 휴가 기능만을 사용하여 동료의 일정을 공유하고 협업을 장려하며 근로시간을 관리합니다. 더 자세한 설명을 위해, 도입 문의를 남겨주세요.

  • API 혹은 다른 서비스들과의 연동을 제공하나요?

    네. 피플허브는 정교한 Open API를 제공하며, Slack, 구글 캘린더, Microsoft Teams, Outlook, 그리팅, 나인하이어, 사람인 리버스 등의 서비스들과의 연동도 제공하고 있습니다. 도움이 필요하실 경우, 도입 문의를 남겨주세요.
